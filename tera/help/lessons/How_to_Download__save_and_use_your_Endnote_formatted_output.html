How to Download, save and use your Endnote formatted output

Download

The easiest is to
1: Right-click on the link and
2: "Save Target As... " in a place you will remember.

 

You can change the default file name to your choice: our example: testme.txt

Open Endnote

In Endnote, open either one of your databases, or a new one.
Then select the Menu Option: File | Import...

A new menu option open
1: Locate your saved file and
2: Choose the Import Option "ISI-CE" in the drop down list

Click on Import... the data import should start.
