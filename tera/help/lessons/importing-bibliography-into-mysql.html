Preliminary Settings in ENDNOTE:

In Endnote, Select Edit > Preferences > Folder Location and choose the relevant folder for Styles for eRA; This is where we maintain the common copy for eRA-tab delimited which we use for exporting the library.

At the moment, we use \\salt\e-RA\Bibliography\Styles-copy (CHECK)

Open the Style manager to add era-tabdelimited to the list of your prefered output styles. Once you have connected your style folder in the step above, eRA-tabdelimietd should be available in the long list of stles available.

Tick all teh styles you are happy to use in endnote (including the eRA-Tabdelimited,. Uncheck teh styles you do not need.

Export the library as a tab delimited file

Select the output style you need (here Era tab delimited)

Select all the references (CTRL A)

Choose the File > Export menu option

Choose a folder and filename you will remember, and choose save as type txt

Open the Tab Delimited file in Exel

Using Exel, open the saved text file: as it is a text file, it opens the import wizard. Fill in all the relevant options to import the file correctly into exel.

Perfomr a few checks to ensure your data is correct

Save the file as ODS
