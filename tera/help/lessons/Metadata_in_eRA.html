How to make a page from old web site page:

Go to that page and right click: and select the menu option: View Source

Select what you need for your page. and CtrlC (Menu Edit | Copy)

Paste your clipboard in Context or any Text Editor.
and edit what you need.

Example: Please add the </p> at the end of paragraph
and </LI> (at the end of <li> lines.

Every <tag> should be closed with the corresponding </tag>

In the text you want to have a link to that page, just insert

 

<a href="index.php?area=home&page=index&dataset=2&sub=restest" target="out">Name of the page...</a>

where dataset is the datasetID of your set of data and restest the page name without the extension (php)
